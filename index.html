About Me

I’m Omar Osama

4th year bachelor’s student majoring in Computer Science and Engineering at the German University in Cairo.

I started programming in 2020 soon after graduating high school at Al Wadi International school. Since then I've developed a great passion for programming and an eagrness to always learn and discover new concepts to expand my knowledge.

I started programming with Python where I learned some basic fundamentals of programming and did a couple of mini projects, I then moved on to learning a more strongly typed programming language like Java where I learned and am still learning much advanced programming concepts and did some advanced projects.

While learning Java I took a dive into other programming paradigms like logical and functional paradigms by learning Prolog and Haskell and also doing some projects in each of these languages.

I then started learning about Databases and SQL. Initially, I was introduced to these fundamental concepts as part of my academic curriculum, gaining a solid foundation in database management and SQL query language. However, my passion for this field led me to delve deeper independently. I embarked on a self-guided learning path, devouring books, online courses, and practical exercises to expand my knowledge. As my expertise grew, I started exploring advanced database concepts, from normalization and indexing to stored procedures.

I still look forward to learning even more programming concepts and languages and would definitly love to apply the skills and knowledge I've learned in the real world to take on a new challenge.
